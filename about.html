I am a passionate Flatiron School software engineer and social entrepreneur. I am proud of my beginnings as a youth developer with inner-city kids through then through education within the halls of congress and now in software engineering.

As a software engineer with a focus on React and Rails, I've built a robust portfolio including a professional coffee cupping app (Catoro.now.sh). I've also built custom projects library projects that have been used widely by the developer community (https://rubygems.org/gems/coingecko).

Tech Projects

  • Yijing Ball Z: I-Ching oracle inspired by Dragon Ball Z. Vanilla JavaScript frontend and developed a Rails API backed with endpoints for readings and users. Handled AJAX CRUD calls in JSON. Used Facebook JS SDK.
